Passo a passo

Tela Inicial

-Botão +: Permite iniciar criação de novo arquivo.
-Botão lupa: Permite pesquisar os arquivos salvos.

Tela de registro

-Botão Salvar: Permite salvar o arquivo para pesquisa/exclusão posterior.
-Botão Cancelar: Apaga todo o conteúdo digitado.
-Botão Itálico: Converte todo o texto da tela para itálico.
-Botão Negrito: Converte todo o texto da tela para negrito.
-Botão Regular: Restaura o texto da tela para seu valor padrão.
-Seleção de fonte: Permite selecionar o tamanho da fonte.

Tela de pesquisa

-Permite visualização dos textos salvos tanto como exclusão.
